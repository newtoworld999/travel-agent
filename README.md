# Travel Agent Chatbot

A simple Python chatbot that acts as a travel agent. It uses a list-based memory to store conversation history, summarizes the chat after every 5 exchanges, and replaces older messages with the summary to keep context small.

## Features

- Streams responses token-by-token
- Maintains conversation history in a Python list
- Summarizes chat history after every 5 conversations
- Replaces older history with the summary to reduce context size
- Uses a prompt template for travel-specific behavior
- Responds exactly with `I can’t help with it.` for non-travel questions

## Files

- `travel_agent.py` — main chatbot script
- `output.txt` — example log/output file generated by the demo or chat session
- `README.md` — project documentation

## Requirements

- Python 3.8+
- Optional: a working LangChain/Ollama environment

## Usage

### Interactive mode

```bash
python3 travel_agent.py --output output.txt
```

Type travel-related queries and type `exit` to quit.

### Demo mode

If the project uses a fallback implementation or if you want to test use cases automatically:

```bash
python3 travel_agent.py --demo --output output.txt
```

This will run the built-in demo flow and save logs to `output.txt`.
